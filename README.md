## Instalation Guide
### Prerequisites

-   Java 8
-   Maven 3.8
-   Git
### Steps
1.  Use Git to clone the project

```git clone https://github.com/albravoc68/Pokedex```

3.  In the project folder use maven to install it

```mvn clean install```
    
3.  Start Project

```java -jar target/Pokedex-1.0-SNAPSHOT.jar```
